# 文件 / 目录选择 Demo

基于 **Vue 3 + Vite 7 + Express 5** 的前后端小项目，纯后端读取文件系统 + 自建文件选择弹窗，**可获取绝对路径**。

## 技术栈

| 层   | 技术       | 版本    |
| ---- | ---------- | ------- |
| 前端 | Vue + Vite | 3.5 / 7 |
| 后端 | Express    | 5.2     |

## 核心设计

不依赖浏览器 File System Access API，而是通过后端 Node.js 直接读取文件系统，前端自建文件选择弹窗：

- **后端**：Express 提供 `/api/fs/*` 接口，读取本地文件系统目录内容，返回绝对路径
- **前端**：自建 `FilePickerModal` 弹窗组件，带搜索、导航、多选等功能

> 此方案可获取文件的绝对路径，无浏览器兼容性问题。

## 功能

- 📄 选择文件 — 独立按钮，弹窗选择文件
- 📂 选择文件夹 — 独立按钮，弹窗选择文件夹
- 🔍 搜索 — 实时模糊搜索当前目录内容
- ⬆ 返回上级 — 导航到父目录
- 🏠 快捷访问 — 家目录、桌面、文档、下载
- 💾 盘符导航 — 左侧栏列出所有磁盘盘符
- ✅ 多选 — Ctrl+点击选择多个项目
- 📍 绝对路径 — 返回完整的本地路径

## 快速启动

```bash
# 1. 安装后端依赖
cd server && npm install

# 2. 安装前端依赖
cd ../client && npm install

# 3. 启动后端（端口 8642）
cd ../server && npm run dev

# 4. 启动前端（端口 7891）— 另开终端
cd client && npm run dev
```

然后在浏览器打开 http://localhost:7891

| 服务 | 端口 |

## API 端点

| 方法 | 路径            | 说明                        |
| ---- | --------------- | --------------------------- |
| GET  | /api/health     | 健康检查                    |
| GET  | /api/fs/home    | 获取用户家目录路径          |
| GET  | /api/fs/drives  | 获取可用磁盘盘符列表        |
| GET  | /api/fs/list    | 列出目录内容 (?dir=xxx)     |
